## 实现内容

### Phase 1: 项目骨架与设计系统
- Vite + Vue3 + TypeScript 项目初始化

### Phase 2-4: API 层 + 状态管理 + 路由
- API 层（11 文件）：request.ts（Axios 拦截器剥离外层）+ sse.ts（fetch+ReadableStream 流式）+ 9 个业务模块
- Pinia stores（4 文件）：auth / chat / resume / app（全部 Composition API）
- 路由：6 个业务路由 + 登录守卫

### Phase 5-8: 14 个业务组件
- 通用：EmptyState（SVG 几何插画）/ LoadingOverlay
- 简历：ResumeCard（衬线姓名+收藏星标）/ FilterBar / UploadDialog / ResumePreview
- 对话：MessageBubble / StreamIndicator（脉冲动画）/ SessionList / ChatPanel（SSE 全流程）
- 候选人：CandidateCard / CandidateCompare（ECharts 雷达图）/ TagInput
- 看板：ChartWidget

### Phase 9: 8 个页面视图
- Login.vue：杂志封面级登录页（左侧深墨绿品牌区 + 右侧表单区，Fraunces 64px 标题）
- Layout.vue：深墨绿侧边栏 + 折叠 + 用户下拉
- Workbench.vue：三栏工作台（会话列表+对话面板+候选人推荐）
- ResumeList.vue：简历库（筛选+上传+导出+卡片网格+分页）
- ResumeDetail.vue：详情页（时间线+预览+标签+面试评价）
- Dashboard.vue：数据看板（4 统计卡 + 3 ECharts 图表）
- JdMatch.vue：JD 匹配（textarea + 结果展示）
- Settings.vue：SMTP 配置

### Phase 10: 测试与构建
- 21 个测试文件，106 个测试全部通过
- 构建验证通过，dist 产物生成

## 技术要点
- Axios 拦截器统一剥离 {code,message,data} 外层，业务代码只处理 data
- SSE 用 fetch + ReadableStream 实现，支持 POST + 8 种事件分发
- 路由守卫：未登录跳 /login，token 失效(1002)自动登出
- ECharts 雷达图与柱状图/饼图，配色与设计系统一致
- 全局 grain 纹理 + 页面切换动画 + 卡片 hover 微交互
- 严格 TypeScript，禁止 any（除 ECharts option）
